---------------------------------------------------------------------------
Crear categoría:
---------------------------------------------------------------------------
http://localhost:3000/api/categorias

Body JSON:

{
  "nombre": "Matemática"
}



---------------------------------------------------------------------------
Crear subcategoría:
---------------------------------------------------------------------------



POST →

http://localhost:3000/api/subcategorias

{
  "nombre": "Álgebra",
  "categoriaId": "ID_DE_LA_CATEGORIA"
}


---------------------------------------------------------------------------
Crear nivel:
---------------------------------------------------------------------------



POST →

http://localhost:3000/api/niveles

{
  "nombre": "Fácil"
}




---------------------------------------------------------------------------
Crear rango de edad:
---------------------------------------------------------------------------



POST →

http://localhost:3000/api/rangos

{
  "descripcion": "5-7 años",
  "edadMin": 5,
  "edadMax": 7
}
